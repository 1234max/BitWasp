admin_set_progress(int $order_id, int $progress) : boolean

Admin Set Progress

This function is used by autorun jobs to set the progress of the order $order_id to $progress. Unlike the normal progress_order() function, which requires the current progress and calculates the next progress number accordingly, this function can arbitrarily set an order to any stage in the order process.
